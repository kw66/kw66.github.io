# kw66.github.io

Root GitHub Pages site for the academic homepage of Xulin Li.

## Notes

- This repository is intended for the root user site: `https://kw66.github.io/`
- Existing project pages like `https://kw66.github.io/PhD_Simulator/` can continue to coexist under subpaths
- Main entry file: `index.html`

## Publish

1. Create or use the repository named `kw66.github.io`
2. Push this directory to the repository default branch
3. GitHub Pages will serve the root site from `https://kw66.github.io/`
